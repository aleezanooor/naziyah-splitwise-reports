Naziyah Splitwise Reports

A personal, non-commercial connector that reads Splitwise account data so Naziyah can prepare private expense summaries, balance reports, and spending insights for the account owner.

Purpose

This application is built for personal use. It connects to Splitwise through the official API with the account owner's explicit authorization, then uses the resulting data to answer questions such as recent spending by group, outstanding balances, monthly totals, and settlement history.

Data Use

  • Splitwise data is used only to generate private reports for the authorized account owner.
  • The connector does not sell, publish, or share Splitwise data.
  • API credentials and access tokens are treated as confidential secrets.
  • Stored data, if any, is limited to what is needed for local reporting and debugging.

Project Status

This is a private personal project, not an official Splitwise product. Splitwise has not reviewed, endorsed, or certified this connector.

Support

For questions about this connector, contact the repository owner through GitHub. For Splitwise account or product support, use Splitwise support.
